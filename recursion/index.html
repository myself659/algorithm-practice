recursion

递归是一种top-down的思维方式。

A recursive function is a function that calls itself until a “base condition” is true, and execution stops.

递归体现了代码的利用。

A function calling itself.

分解与组合要正确

steps

以常见的fib问题为例。

function fib(n) {

}

step-1 理解问题定义问题函数

int fib(int n) {

}

也就是三个问题：

  1. 函数名
  2. 函数入参
  3. 函数返回值

step-2 找到子问题 （分解成子问题）

重要公式：

fib(n) =   fib(n-1) + fib(n -2)

题中是将fib(n)分为两个两个问题：fib(n-1)，fib(n -2)

int fib(int n) {

  int prefib = fib(n-1);
  int preprefib = fib(n-2);
}

step-3 应用子问题 （组合子问题）

int fib(int n) {

  int prefib = fib(n-1);
  int preprefib = fib(n-2);

  return preprefib+ prefib; 
}

将两个子问题组合起来。

step-4 递归结束的条件及最基本的子问题

int fib(int n) {
  if(0 == n) return 1;
  if(1 == n) return 1;

  int prefib = fib(n-1);
  int preprefib = fib(n-2);

  return preprefib+ prefib; 
}

step-5 整个过程的可视化，加强理解

调用与返回的整个过程。
